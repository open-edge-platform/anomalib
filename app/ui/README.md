# Geti Inspect

## Setup

Install the dependencies:

```bash
npm install
```

## Get started

Start the dev server:

```bash
npm run start
```

Build the app for production:

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```

## Testing

Use the following commands to test your changes.

```bash
npm run format
npm run lint
npm run type-check
npm run test:unit
npm run test:component
```
